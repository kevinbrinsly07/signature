## Demo Features
The demo showcases all features of SignMe:

### Document Signing Demo
1. Navigate to the Document Signing tool
2. Draw a signature on the canvas
3. Try different pen thicknesses and colors
4. Use the eraser and undo functions
5. Apply signature enhancement
6. Export as PNG

### Stamp Designer Demo
1. Switch to the Stamp Designer tool
2. Select different shapes (circle, rectangle, oval, triangle)
3. Adjust dimensions and colors
4. Add text with various fonts and effects
5. Manage layers
6. Apply enhancement styles (Original, Vintage, Fadeout)
7. Export your custom stamp
