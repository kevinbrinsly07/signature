# SignMe Changelog

## Version 1.1 (December 2025)
- **New Features:**
  - Added vintage and fadeout enhancement styles for stamps
  - Improved signature smoothing algorithm
  - Enhanced mobile responsiveness
  - Added keyboard shortcuts for better usability
- **Improvements:**
  - Optimized canvas performance for larger signatures
  - Better color picker interface
  - Improved layer management in stamp designer
  - Enhanced export quality
- **Bug Fixes:**
  - Fixed eraser tool behavior on touch devices
  - Resolved canvas clearing issues in some browsers
  - Fixed text positioning in stamp designer
- **UI/UX:**
  - Modernized interface with Tailwind CSS
  - Added loading states and better feedback
  - Improved navigation between tools

## Version 1.0 (Initial Release)
- Initial release of SignMe
- Document Signing tool with canvas drawing
- Stamp Designer with basic shapes and text
- Export functionality for PNG files
- Responsive design for desktop and mobile
