SignMe User Guide

Version 1.1

Overview

SignMe is a comprehensive web-based suite that provides two powerful tools:

  • Document Signing: Create professional digital signatures
  • Stamp Designer: Design custom stamps with various shapes and effects

You can switch between these tools using the navigation in the header.

Document Signing Tool

Getting Started

1. Access the Tool
Open index.html in your web browser. The Document Signing tool loads by default.
2. Drawing Your Signature
Click and drag on the white canvas to draw your signature. The canvas responds to both mouse and touch input.

Drawing Tools

  • Pen Mode: Default drawing tool with adjustable thickness and color
  • Eraser Mode: Remove parts of your signature
  • Pen Thickness: Choose from Thin, Medium, Bold, or Thick presets
  • Color Picker: Select any color for your signature

Canvas Controls

  • Undo (Ctrl+Z): Remove the last stroke
  • Clear (Del/Backspace): Reset the entire canvas
  • Fullscreen: Expand canvas for detailed work
  • Enhance: Smooth and professionalize your signature

Exporting Your Signature

1. Click "Save Signature" to open the preview modal
2. Review your signature in the preview
3. Click "Download PNG" to save as a transparent PNG file
Tip: PNG files with transparency work well in documents and can be easily imported into PDF editors, word processors, or design software.

Stamp Designer Tool

Getting Started

1. Access the Tool
Click "Stamp Designer" in the navigation or open stamp-designer.html directly.

Creating a Basic Stamp

1. Select Shape
Choose from Circle, Rectangle, Oval, or Triangle
2. Set Dimensions
Adjust width and height using the sliders (20mm to 350mm)
3. Choose Style
Select filled or hollow shape, and set colors

Adding Text

1. Enter Text
Type your desired text in the text input field
2. Customize Appearance
Select font, size, color, and curve level
3. Add to Stamp
Click "Add Text" to place the text on your stamp

Managing Layers

Stamps can have multiple elements. Use the layer panel to:

  • Move elements up or down in the stacking order
  • Delete unwanted elements
  • Select elements for editing

Enhancement Styles

Apply realistic effects to your stamp:

  • Original: Clean, standard stamp design
  • Vintage: Aged appearance with paper texture and weathering
  • Fadeout: Worn effect from repeated use

Exporting Your Stamp

1. Click "Export PNG" to open the enhancement preview
2. Choose a style (Original, Vintage, or Fadeout)
3. Click "Export Selected" to download your stamp

Troubleshooting

Signature Looks Jagged

Use the "Enhance" button to smooth your signature, or try drawing with a thicker pen setting.

Text Not Appearing Correctly

Some fonts may not render consistently across browsers. Try a different font or check browser compatibility.

Canvas Not Responding

Ensure you're using a modern browser with HTML5 Canvas support. Clear your browser cache if issues persist.

Warning: Enhancement effects are applied during export and cannot be undone. Always preview before downloading.
